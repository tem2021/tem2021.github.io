A.颜色

原主题这非常多的地方运用了绿色橙色等，因此如果决定好了颜色，仅仅在mastery.css中更改是远远不够的，建议使用ctrl+shift+r进行全局搜索并更改。

补充说明：可以进入网站，点击不同按钮进入不同页面，这样就能看到许多意想不到的颜色。

例如：开始阅读这个按钮是制作了css动画的，最后颜色会变为绿色

这里就有几个工具可以推荐：

a.截图工具Snipaste
这个软件可以运用到许多地方（比方说我之前之所以会知道这个软件主要是因为板绘需求）具体内容就不多赘述了，可以自行上网搜索了解

b.Utools
这个工具可以说是运用面也很广，不过目前我只用这个进行取色与色彩识别

#有关色彩识别


Utools内的色彩提取精度很高，很多时候会出来一堆颜色，反而不利于我们的工作，因此这里附上另一个网站：在线图片颜色取色器
-极念网 (jiniannet.com)

如图所示，省略相近颜色这个选项可以帮助我们排除一些多余选项

作为半个艺术生，这里在附上一个使用下来效果上乘的颜色网站：十六进制颜色代码表，图表及调色板
- Encycolorpedia

在网站中使用了大量图片，这些大都可以在hexoblog\themes\hexo-theme-matery\source\././blog-making-journals中找到，参照上述文章的方法可以很容易完成更改

下载这个插件，就可以使得用word编写的文本转化为md格式，还是非常好用的

2.上传图片

参考网站:
解决Hexo博客图片无法显示_哔哩哔哩_bilibili
hexo博客中如何插入图片_不作声的博客-CSDN博客
首先需要下载两个插件

 npm install hexo-renderer-marked

接下来在hexoblog下的_config.yml文件中做一下改动

post_asset_folder: true
marked:
  prependRoot: true
  postAsset: true

在source/_posts文件夹中把已经写好的文章放入/新建一个文章（一定要相同名称）

hexo cl
hexo g

hexo d (上传部署)

如果出现本地预览和部署结果不同的情况，有一下两种可能：
1.没有清楚缓存，建议重新执行 hexo cl && hexo g
2.也就是我遇到的情况，网页中的背景、icon都无法加载出来，此时可能是浏览器本身的问题，在blog首页执行shift+F5即可
